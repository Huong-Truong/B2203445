## ⚙️ Cài Đặt

1. Clone repository:
    ```bash
    git clone https://github.com/Huong-Truong/B2203445.git
    ```

2. Cài đặt phụ thuộc:
    ```bash
    composer install
    ```

3. Thiết lập môi trường:
    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4. Chạy migration:
    ```bash
    php artisan migrate
    ```

5. Khởi động server:
    ```bash
    php artisan serve
    ```
